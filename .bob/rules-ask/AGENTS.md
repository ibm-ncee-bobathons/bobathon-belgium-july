# AGENTS.md

This file provides guidance to agents when working with code in this repository.

## Documentation Context (Non-Obvious)

- **`wxo-implementation-guide.md`** in Lab 2 is the canonical ADK reference — the inline comments in code may be outdated relative to it.
- Lab 5's `src/converter.py` is **intentionally incomplete** for test coverage — 64% coverage at the end of Lab 5 is expected; Lab 7 asks students to close the gap.
- `Lab 1/starter/` contains the incomplete scaffold students work from; `Lab 1/solution/` is the reference answer — always clarify which you are looking at.
- The `.bob/rules/wxo-development.md` in Lab 2 is a Bob rule file used as lab content/example, not a project-level rules file for the whole repo.
- Lab 4 has no source code — it is entirely a link to an external browser-based interactive demo.
- "ADK" refers to the watsonx Orchestrate Agent Development Kit, not Android or any other SDK.
